What you'll learn

We'll answer common questions like:

  • What is Continuous Quality?
  • How can Continuous Quality be implemented in a real scenario using GitHub?
  • How can UI & BDD tests be run on agents?
  • How can failure of UI & BDD tests in GitHub create bugs in Azure Boards?

And when you're done you'll be able to:

  • Understand Continuous Quality strategy
  • Make use of GitHub actions to run BDD & UI tests against web application.
  • Configure the GitHub actions in such a way, that failure of UI or BDD tests should result in creation of bugs in Azure Boards.

What you'll build

In this Lab, we'll be walking through the Continuous Quality practices and how we can implement these using GitHub.

Prerequisites

Before you start, you should be familiar with GitHub and Continuous Quality and must have completed the previous labs. We recommend taking a look to the following resources and Learning Labs prior to this course:
